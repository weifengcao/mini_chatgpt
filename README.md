# mini_chatgpt
This is to create a mini chatGPT, through which help user understand how to integrate an AI agent into a chat service

I plan to design a platform to provide an ai first chat service like chatgpt. Each company has a super account. for their agent registration, with the company, their employees can log in the chat service for chatting, each chat session will serve 100 users concurrently, each company can registers up to 20 agents. each chat session will call one single agent for the users to use. our platform can serve up to 100 companies per day. latency for chatting is < 1000ms, availability is 99%, the agent asscoiated with the chat session can also interact with users and execute user's tasks. It is like a chatgpt, chatting, runnig tasks, give response etc.
